# SpotifyGPT

SpotifyGPT is a foundation repository for exploring AI-assisted workflows around Spotify data.

## Getting started

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
pytest
```

## Development

- Source code lives in `src/spotifygpt`.
- Tests live in `tests/`.
